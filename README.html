PIXAR USD Syntax Highlighter

Version 0.3 Released 2019-12-29
by Andrew Hazelden

Overview

This project hosts a collection of syntax highlighting modules for Pixar USD (Universal Scene Description) ASCII format documents.

Version History

Version 0.3 - 2019-12-29

  • Added a GeSHi module with two example php webpages that render USD ASCII files a light and dark background CSS variation.

Version 0.2 - 2016-01-21

  • Added support for highlighting files with the .usda file extension

Version 0.1 - 2013-12-13

  • Created the initial USD syntax highlighter module for Notepad++
